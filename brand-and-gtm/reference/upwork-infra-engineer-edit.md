**Updated Description (paste this in):**

We're Tryps — a group travel planning app launching May 2, 2026. You add a travel agent to your iMessage group chat, text it your trip idea, and it plans the whole thing. Think Partiful but for travel.

We're building a 3-person brand and go-to-market team under our creative director. We need an AI and automation engineer who builds every tool, workflow, and system our GTM operation runs on. You don't create content — you create the systems that make everyone else 10x faster.

**What this role looks like:**

Our creative director calls the shots on what content gets made. A growth engineer handles day-to-day campaigns. Your job is to build the infrastructure underneath both of them — trial and deploy the right tools, wire them together with automation, build dashboards so we know what's working, and create AI-powered pipelines that turn one piece of content into dozens of variants.

You live in the terminal. Claude Code or Cursor is your primary working environment. You're the person who builds a Figma plugin in 45 minutes, wires up an n8n workflow that auto-pulls social metrics into a dashboard, and deploys a Remotion video template pipeline so the team can produce product demos in minutes instead of hours.

**What you'll build (across our 13 GTM scopes):**

- Scheduling & publishing infrastructure — set up Buffer/Metricool across 8 platforms, build auto-repost workflows to secondary platforms
- Creator tool stack — trial and deploy influencer discovery/verification tools (Stormy AI, Modash, HypeAuditor), build affiliate link system with iMessage deep links (Branch), build ambassador stats dashboard
- Video template pipeline — build Remotion templates for product demo clips (screen recording in, branded 15-sec clip out with transitions and callouts)
- Ad variant pipeline — build a Figma plugin for bulk ad variant generation (the Austin Lau / Anthropic model), set up Recharm for auto-generating Meta ad variants from UGC clips, install Meta pixel and conversion tracking
- SEO/AEO infrastructure — implement schema markup and structured data on jointryps.com, build 3-5 SEO landing pages, set up AEO monitoring
- Referral system — build deep link redirect pages (jointryps.com/go/{creator}), wire Branch attribution, build referral tracking
- Email setup — deploy Loops for onboarding drip sequences, build waitlist email capture
- Analytics & automation — build all n8n/Make workflows (metrics → dashboards, alerts → Slack, payments → reminders), set up Looker Studio dashboards, configure LangFuse for agent observability, build investor-ready cost tracking
- App Store — set up ASO tools, configure Apple PPO testing, prep Product Hunt listing

**After week 2-3, you shift from building to maintaining + improving.** The initial sprint is standing up all systems. After that, you improve them based on what the team needs, fix what breaks, and build new tools as new needs emerge.
