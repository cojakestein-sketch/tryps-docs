# Path to the Plane

**Concept:** Revenue-to-lifestyle model showing what Jake's travel life looks like at each user milestone. Not for investors — this is Jake's personal north star and internal motivation deck.

**Data & Analysis:** `data/path-to-the-plane.md`

## The Four Tiers

1. **First Class Everywhere** — Lie-flat international, suites on Emirates, lounge life. ~$150K/year.
2. **NetJets Card** — Skip TSA forever, text your pilot. ~$500K/year.
3. **Own a Plane + 6 Friends** — Challenger 350, take the boys to Tulum on Tuesday. ~$4.8M/year.
4. **The Fleet** — Two G650s, a midsize for quick hops. Send anyone anywhere. ~$26M/year.

## Key Insight From Pressure Test

Marty's original model was essentially the bull case. The realistic base case requires **3–4x more users** for each tier, primarily because:

- Blended take rate is 8%, not 12% (flights are low-margin)
- Paid conversion is 15%, not 25% (planning-to-booking gap)
- Operating margins are 20%, not 35% (at sub-$50M revenue)

**Base case user requirements:**
- First Class: **179K users** (not 34K)
- NetJets: **595K users** (not 112K)
- Own Plane: **5.7M users** (not 1.08M)
- The Fleet: **30.9M users** (not 5.8M)
