## What

Engineering quality — does the code work? QA pass on 19 core flows, formal criteria validation (240+ criteria across all scopes), ClickUp status reconciliation, and regression testing on all integrations.

## Why

QA is continuous, not a phase. It runs from Day 1 through April 2, with intensity increasing as scopes finish. Without formal validation, we ship bugs to beta users.

## Success Looks Like

- All 19 core flows validated with pass/fail rubric
- 240+ criteria across all scopes tested and passing
- ClickUp statuses reconciled to reflect actual state
- Regression testing on iMessage-app sync, Stripe, connectors
