## 1. Tools — Download Now

**Action required for everyone. Get both of these installed before standup.**

### Wispr Flow

Wispr Flow is a voice-to-text tool that lets you dictate anywhere on your Mac or phone — it transcribes in real-time with near-perfect accuracy. You'll use it to answer your 5 daily questions via voice instead of typing. Jake has invited the full team.

- Download + sign up: [wispr.flow](https://www.wispr.flow/)
- Test it by answering today's questions (Section 3)

### Claude Code Agent Teams

Agent teams let you deploy multiple Claude Code instances that work together in parallel — each one investigates a different hypothesis and they challenge each other's findings before converging on an answer. Think of it as a research swarm you can spin up from your terminal.

- Docs: [code.claude.com/docs/en/agent-teams](https://code.claude.com/docs/en/agent-teams)
- Enable it: set `CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS` in your Claude Code settings

**Example prompts to try:**

> "Investigate why the deep link flow drops users after OTP — spin up 3 teammates, one looking at the auth middleware, one at the mobile routing, one at the iMessage callback chain"

> "Research the best approach to building a memory architecture for an AI travel agent — have each teammate explore a different pattern (vector DB, structured JSON, conversation log) and compare trade-offs"

> "Find the root cause of the crash on trip detail screen — competing hypotheses: state management race condition vs. missing null check vs. stale cache"

The agent is functional end-to-end for core flows. What's working 100% right now:

- **Expense tracking, voting, and polling** — integrated, not fully regression-tested yet but functional
- **Trip queries** — itinerary, balances, and participants all working
- **Activity creation and activity suggestions** — working

What still needs work: polling reminders and making sure polls work end-to-end, trip completeness levels, routing logic design doc, Jennifer Test validation (needs implementation), and the cross-scope interface with Rizwan's agent-intelligence — I need to work with him on that, hopefully today.

**4. What's stuck? Do you need anything from Nadeem, Rizwan, or Jake?**

**5. Anything surprising?**

The big surprise was that all media links from Linq (like if someone sends a photo to set as the trip background) expire after one hour. I had to build a pipeline that copies the image content, re-uploads it to Supabase Storage, and then uses our hosted URL in the trip. Didn't expect that.

Also, the HMAC signature validation from Linq is inconsistent — it works in some cases and gets skipped in others. That's a security gap I'm addressing today.
