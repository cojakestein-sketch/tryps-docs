## Intent

Let users connect their entire travel life — airlines (United, Delta, AA loyalty), hotels (Marriott, Hilton, Airbnb), and even ChatGPT with a custom prompt so we can receive travel context. A connector framework using OAuth + API key patterns lets us import existing bookings and enrich the trip experience.

## Key Areas

- Airline account linking (loyalty numbers)
- Hotel loyalty linking (Marriott, Hilton, Airbnb)
- ChatGPT custom prompt integration (user sends travel context, we receive it)
- Connector framework (OAuth + API key patterns)
- Import existing bookings from connected accounts
