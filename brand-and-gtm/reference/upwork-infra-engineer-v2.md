**Category:** Web, Mobile & Software Dev > Automation

**Description:**

We're Tryps — a group travel planning app launching May 2, 2026. You add a travel agent to your iMessage group chat, text it your trip idea, and it plans the whole thing. Think Partiful but for travel.

We have a creative director who calls the shots on content. We have a growth engineer who produces content and runs campaigns. We need you to build every system, workflow, and tool they run on.

This is a founding role. You're not filling a seat — you're building the entire marketing infrastructure from zero. If you do this well, you grow with the company. Think of this as head of GTM engineering.

**What you'll build in the first 2 weeks:**

- Set up Buffer or Metricool across 8 social platforms. Connect all accounts, configure auto-repost workflows to secondary platforms.
- Install Meta pixel on jointryps.com. Configure conversion events (app install, trip created).
- Set up Loops for email. Build waitlist email capture on the landing page. Configure welcome email.
- Trial 3-5 creator discovery tools (Stormy AI, Modash, HypeAuditor). Run test searches, recommend one.
- Build affiliate link redirect pages (jointryps.com/go/{creator} → iMessage deep link). Deploy Branch for attribution.
- Set up ASO.dev for App Store keyword research. Configure Apple PPO testing.
- Wire your first n8n automation: new App Store review → Slack alert.

**What you'll build in weeks 3-4:**

- Build Remotion templates for 15-sec product demo clips (screen recording in, branded clip out).
- Build a Figma plugin for bulk ad variant generation from UGC clips.
- Implement schema markup and structured data on jointryps.com.
- Build 3-5 SEO landing pages.
- Set up Looker Studio dashboard pulling data from Buffer, Meta, and AppsFlyer.
- Configure LangFuse for AI agent observability.
- Build UTM naming system and cost tracking.

**After launch, you maintain and improve.** Fix what breaks, add new automations as needs emerge, build whatever the creative director or growth engineer needs next.

**Required skills:**

- Python or TypeScript (you build across both)
- n8n, Make, or similar automation platforms (not just Zapier)
- API integrations — you'll connect 10+ marketing tools together
- Web development basics — HTML/CSS for landing pages, OG images, schema markup
- Claude Code or Cursor — you live in the terminal, AI tools are your daily workflow
- Comfortable evaluating and deploying unfamiliar SaaS tools fast

**Nice to have:**

Buffer or Metricool (social scheduling), CapCut + Remotion (video), Canva (graphics), Branch + AppsFlyer (attribution), Loops (email), Meta Ads Manager (paid), ASO.dev (App Store), Ahrefs (SEO), n8n (automation), Looker Studio (dashboards), LangFuse (AI observability), Slack (alerts).
