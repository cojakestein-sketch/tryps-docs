# P3 Scope 1: Vote on My Behalf — Spec

> **Gantt ID:** `p3-vote-on-behalf`

## Intent

The agent learns your travel preferences from Travel DNA + vote history, then votes for you when you don't respond. Before the 48-hour voting deadline, the agent implies what it thinks you'd pick — you can change your vote or let it stand. If the countdown runs out, the agent's inferred vote counts.

## Key Areas

- Agent learns user preferences from Travel DNA + vote history
- Auto-vote inference engine (predict vote before 48h deadline)
- Pre-deadline nudge: "I think you'd pick Option B — change it?"
- 48-hour auto-vote fallback (agent votes if user doesn't)
- Confidence scoring (high/medium/low certainty)
- Opt-in/opt-out per trip and per category (activities, dates, locations, other)
