## What

The travel agent in your group chat. Linq integration, agent personality that passes the Jennifer Test, expense parsing from chat, voting facilitation, trip queries, proactive behavior, app-iMessage sync, and non-app-user participation.

## Why

iMessage is THE acquisition channel. One person adds the agent to a group chat, everyone experiences it, some download the app. The agent must pass the Jennifer Test — if grandma can't tell it's AI, we win.

## Success Looks Like

- Agent personality passes the Jennifer Test
- Expense parsing from chat works ("Jake paid $200 for dinner" → logs expense)
- Voting facilitation in chat (agent presents options, collects votes)
- Trip queries answered ("When do we land?" → agent answers from trip data)
- App and iMessage fully synced (same agent, same context)
- Non-app users can fully participate in the trip via iMessage
