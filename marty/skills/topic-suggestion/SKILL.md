# Topic Suggestion

At 3pm ET daily, DM Jake on Slack with a suggested topic for tonight's deep-dive 1-pager.

## Steps

### 1. Gather Context

- Read Jake's Google Calendar for tomorrow's meetings (use `gog cal list`)
- Read current ClickUp sprint status (what phase are we in, what's stuck)
- Read the last 3 daily memory files for recent themes and conversations
- Read `tryps-docs/shared/priorities.md` for current focus areas

### 2. Pick a Topic

Choose a topic that is:
- **Relevant to this stage of the business** (pre-launch, building, fundraising, etc.)
- **Actionable** — something Jake can apply this week, not theoretical
- **Not something we've already covered recently** (check last 5 nightly reports)

Good topic types:
- Prep for a meeting tomorrow (investor, advisor, partner)
- A framework or strategy relevant to current sprint (e.g., GEO/AEO, brand books, growth loops)
- A competitor or market move worth understanding
- A technical decision the team is facing
- A leadership/management insight relevant to the team's current dynamic

### 3. DM Jake on Slack

Send a **3-line** Slack DM to Jake (`U0AK8FANGNM`):

```
Line 1: The topic (clear, specific)
Line 2: Why it's relevant right now (1 sentence)
Line 3: "Reply with a different topic if you'd prefer, otherwise I'll write this up tonight."
```

Example:
> **GEO/AEO optimization for early-stage consumer apps**
> You mentioned this today and we're about to push brand strategy — understanding how AI search surfaces apps changes how we position.
> Reply with a different topic if you'd prefer, otherwise I'll write this up tonight.

### 4. Handle Response

- If Jake replies with a different topic, use that instead
- If Jake doesn't reply by 7pm ET, use the suggested topic
- If Jake says "skip" or "none", omit section 4 from the nightly report
