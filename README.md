# Tryps Team Documents

Shared documents for the Tryps team — specs, flow diagrams, design references, and planning docs.

## Structure

```
user-flows/          # User flow documentation
  userflow-exec-summary.md    # Executive summary of all entry flows
  userflow-diagrams.html      # Visual flow diagrams (open in browser)
```

## How to Use

- **Markdown files** — read directly on GitHub
- **HTML files** — click "Raw" then open in browser, or clone the repo and open locally
- All team members have write access — feel free to add comments via Issues or PRs
