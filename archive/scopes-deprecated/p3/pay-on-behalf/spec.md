## Intent

The X-402 payment layer that lets agents spend money on behalf of the group. When an agent hits an API that returns HTTP 402 (Payment Required), X-402 handles the micropayment from the Tryps Cash wallet. Cost is logged per trip for future billing.

## Key Areas

- X-402 protocol research & integration design
- Tryps Cash wallet system (per-trip agent execution budget)
- Micropayment flow: agent → HTTP 402 → X-402 pays → result
- Cost logging per trip (track agent spend)
- Budget limits & approval thresholds per trip
